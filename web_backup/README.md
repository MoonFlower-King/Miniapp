# Pocket Ledger (小账本)

React + Vite + TailwindCSS 打造的精美记账应用。

## 运行说明

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 构建生产版本
```bash
npm run build
```

## 功能
- 记一笔（收入/支出）
- 查看收支概览
- 交易明细列表
- 本地数据存储

## 目录结构
- `src/components`: UI 组件
- `src/hooks`: 业务逻辑 Hook
- `src/utils`: 常量和工具函数
